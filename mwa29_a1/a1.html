Interactive Computer Graphics - Assignment 1

Getting Started


Programs for this class will be tested and evaluated in an Ubuntu Unix environment.

Topics:

  • Compiling your first OpenGL program

  • Specifying polygons in OpenGL

  • Modifying your first OpenGL program



Download the code found here to get OpenGL software for displaying a single red triangle.

This has been modified from the software that is available from Prof. Ed Angel of the University of New Mexico.
His OpenGL software web site can be found here.

Modify red_triangle.cpp to create an image in OpenGL that contains 1 red triangle, 1 red quadrilateral and 1 red pentagon.

You should not use the deprecated functions of OpenGL in this or any other programming assignment.

Submit a zip file that contains a folder with your initials and assignment number, for example ak876_a1.
All your files (source files, Makefile and a README) should be inside this folder.

The README should describe your program and how to use it.

The Makefile must contain three targets:

  1. the default target compiles all the code
  2. "run" executes the program(s)
  3. "clean" deletes all the object files and executable files.

The TA should be able to unzip your file, 'cd' into your directory, and type 'make run' to run your program.

Grading Scheme

  1. Your code successfully compiles, runs and displays a red triangle: 6 points
  2. The program also displays a red quadrilateral: 2 points
  3. The program also displays a red pentagon: 2 points
